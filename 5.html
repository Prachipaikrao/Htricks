Ip-Tracer


IP-Tracer is used to track an ip address. IP-Tracer is developed for Termux and Linux based systems. you can easily retrieve ip address information using IP-Tracer. IP-Tracer use ip-api to track ip address.



How to install IP-Tracer

  1. apt update
  2. apt install git-y
  3. https://github.com/rajkumardusad/Ip-Tracer.git
  4. cd IP-Tracer
  5. chmod + x install
  6. sh install or ./install

How to use IP-Tracer

  1. trace -m to track your own ip address.
  2. trace -t target-ip to track other's ip address for example ip-tracer -t 127.0.0.1
  3. trace for more information.

or

  1. ip-tracer -m to track your own ip address.
  2. ip-tracer -t target-ip to track other's ip address for example ip-tracer -t 127.0.0.1
  3. ip-tracer for more information.
